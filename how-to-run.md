# How to Run CodeGuard AI

## Requirements

Make sure you have Python 3 installed on your system.
You can check by running: python --version

## Step 1 - Clone the repository

git clone https://github.com/YOURUSERNAME/codeguard-ai
cd codeguard-ai

## Step 2 - Install dependencies

pip install flask flask-cors

## Step 3 - Start the backend

python app.py

You should see: Running on http://127.0.0.1:5000

## Step 4 - Open the frontend

Open index.html in your browser.

## Step 5 - Use CodeGuard

Paste any Python or JavaScript code into the text area,
select the language, and click Analyze Code.

CodeGuard will show you all issues found with line numbers,
explanations, and fixes.
