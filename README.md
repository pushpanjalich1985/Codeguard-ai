# CodeGuard AI 🛡️

A free, open-source tool that analyzes code for security 
vulnerabilities, syntax errors, logic bugs, and bad practices.
Built for FOSS Hack 2026.

## Problem
Developers rely on AI tools to generate code, but AI-generated 
code often contains serious issues that beginners don't notice.
CodeGuard catches them and teaches you how to fix them.

## Features (In Progress)
- 🔴 Security issues (SQL injection, XSS, hardcoded passwords)
- 🟠 Logic bugs (infinite loops, division by zero)
- 🔵 Syntax errors (missing colons, wrong variable names)
- 🟢 Bad practices (debug prints, no error handling)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
